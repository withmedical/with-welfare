WITH Welfare Mall V2

반영 내용
- 회원가입 기능
- 회원가입 관리자 승인/반려
- 회원가입 필수 항목: 이름, 사원번호, 생년월일, 전화번호, 비밀번호
- 관리자 계정: ID with1905 / PW withm*1905 / 성명 김경진
- 직원 로그인: 이름 + 전화번호 + 개인 비밀번호
- 제주 사계펜션 스텔라동, 솔라동 예약
- 캘린더에 예약된 날짜 표시: 스텔라동 X, 솔라동 X
- 본인 사용 / 직계가족 / 지인 이용 구분
- 본인 및 직계가족 무료
- 지인 이용: 1박 270,000원 기준 50% 할인 금액 계산
- 지인 결제 계좌 안내
- 관리자 페이지에서 계좌번호, 1박 금액, 할인율, 연간 박수 기준 수정 가능
- 직원당 연간 10박 기준 제한
- 중복 예약 방지
- 관리자 승인/반려
- CSV 다운로드

사용 방법
1. 압축 해제
2. index.html 실행
3. 관리자 로그인: with1905 / withm*1905
4. 직원 데모 로그인: 홍길동 / 010-1111-2222 / 1234

주의
이 버전은 서버 없이 브라우저 localStorage에 저장되는 운영형 데모입니다.
실제 운영 시 서버, DB, 파일저장소, SSL, 개인정보 암호화, 백업, 문자/메일 연동이 필요합니다.
